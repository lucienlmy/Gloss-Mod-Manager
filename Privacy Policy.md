# Gloss Mod Manager 隐私政策

**生效日期：2026 年 4 月 24 日**

Gloss Mod Manager（下称“本软件”）是一款桌面端游戏 Mod 管理器。我们遵循“尽量本地处理，按需联网”的原则。本政策说明本软件可能收集、使用、存储和共享的信息。

## 1. 我们收集的信息

### 1.1 本地保存的设置和偏好
本软件会在您的设备上保存部分运行所需或偏好类信息，例如：
- 主题、默认启动页、储存路径
- 下载、导入、管理和 MCP 相关开关
- 调试模式状态
- 随机生成的应用实例 ID
- 其他界面配置和功能偏好

这些数据保存在您的本机，不会因正常使用而自动上传。

### 1.2 您主动授权的第三方账号信息
当您使用 Nexus Mods 登录功能时，本软件会与 Nexus Mods 的授权服务通信，并将授权结果保存在本机。可能保存的信息包括：
- 用户 ID
- 用户名
- 访问密钥 / API key
- 邮箱地址
- 个人主页地址
- 支持者 / 会员标记

除非您主动使用第三方账号登录，本软件通常不会要求您提供姓名、电话、地址或支付信息。这些信息仅用于支持 Nexus Mods 相关功能，例如账号识别、浏览和下载相关 Mod。

### 1.3 使用统计信息
本软件包含基础使用统计，用于改进功能和排查问题。统计上报可能包括：
- 事件名称和事件值
- 应用实例 ID
- 软件版本号
- 界面语言

例如，当您切换游戏时，可能会记录一次对应事件。统计数据会发送到 3DM Mod 站的统计接口。

### 1.4 您的本地文件
为了管理 Mod，本软件会在您指定或选择的目录中读取、写入、复制、重命名或删除文件。相关操作通常仅在您的设备上完成，不会因文件管理本身而自动上传文件内容。

## 2. 我们如何使用这些信息
我们仅将上述信息用于以下用途：
- 提供 Mod 浏览、下载、安装、卸载和管理功能
- 记住您的设置和授权状态
- 完成第三方平台登录和验证
- 发送基础使用统计
- 改进兼容性、稳定性和用户体验

## 3. 信息共享
我们不会出售您的个人信息。

在以下情况下，您的信息可能会与第三方服务交互：
- 您主动使用 Nexus Mods、3DM Mod、Thunderstore、mod.io、CurseForge 或 GameBanana 等平台功能时
- 您主动搜索、浏览或下载 Mod 时，对应的关键词、所选游戏和请求参数会发送到相应平台 API
- 您主动打开这些平台的网页或授权流程时

这些第三方服务会按照它们自己的隐私政策处理数据，我们无法控制其处理方式。

## 4. 存储和保留
- 本地设置和授权信息通常保存在您的设备上，直到您手动清除、退出登录或卸载软件。
- 会话级授权令牌仅用于当前授权流程，并会在您清除授权后移除。
- 使用统计由服务端保存和处理，其保留期限由对应服务决定。

## 5. 您的选择
您可以：
- 在设置中更改或清除本地配置
- 退出或清除 Nexus Mods 授权
- 通过删除应用数据来移除本地保存的信息
- 不使用相关第三方功能，以减少对外部服务的访问

## 6. 第三方服务
本软件可能打开或调用第三方网站与 API，包括但不限于 3DM Mod、Nexus Mods、Thunderstore、mod.io、CurseForge 和 GameBanana。您与这些服务的交互受其各自条款和隐私政策约束。

## 7. 政策更新
我们可能会不定期更新本政策。若有重大变更，我们会通过应用内页面或项目公告进行说明。

## 8. 联系我们
如果您对本隐私政策有任何问题，可以通过项目的 GitHub Issues 与我们联系。
