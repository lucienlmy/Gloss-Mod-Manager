# 正当防卫3 Mod 安装教程


### 配置启动项

在安装了Mod后，需要在steam中，右键属性->通用->启动选项 中添加代码:
```sh
--vfs-fs dropzone --vfs-archive patch_win64 --vfs-archive archives_win64 --vfs-fs
```

Mod才能生效
